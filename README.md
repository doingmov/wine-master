# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Установка

1. **Клонируйте репозиторий:**
```bash
git clone <URL_репозитория>
cd <имя_папки_проекта>
```

2.  **Создайте виртуальное окружение:**
```bash
python -m venv venv
```

3. **Активируйте виртуальное окружение:**

- Windows:
```bash
venv\Scripts\activate
```

- Linux / macOS:
```bash
source venv/bin/activate
```

4. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

## Настройка Excel-файла

По умолчанию проект использует файл ```wines.xlsx``` в корне проекта.

Если вы хотите использовать другой файл, можно указать путь через переменную окружения WINE_FILE:

```bash
export WINE_FILE=/path/to/your/file.xlsx   # Linux / macOS
set WINE_FILE=C:\path\to\your\file.xlsx   # Windows
```

## Запуск проекта

```bash
python main.py
```

- Откройте браузер по адресу ```http://127.0.0.1:8000/index.html```

- Страница с коллекцией вин должна отобразиться полностью.

### Проверка работы

1. На странице должна быть указана правильная продолжительность работы винодельни.

2. Каждая категория (Белые вина, Красные вина, Напитки) выводится с карточками продуктов.

3. Промо-товары отмечены значком «Выгодное предложение».

4. У напитков, где сорт винограда не указан, строчка «Сорт винограда» не отображается.

5. Шаблон автоматически обрабатывает любое количество категорий и товаров из Excel-файла.

### Дополнительные советы

- Перед первым запуском убедитесь, что файл wines.xlsx корректен и находится в корне проекта.

- Можно использовать несколько тестовых Excel-файлов и переключать их через переменную окружения или аргументы командной строки.

- После добавления новых вин или категорий обновляйте страницу в браузере для проверки рендеринга.

- Для разработки удобно включать инструменты разработчика в браузере, чтобы проверять верстку и карточки.
